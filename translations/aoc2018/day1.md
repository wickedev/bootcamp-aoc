## 파트 1

주어진 입력의 모든 숫자를 더하시오.

예)
```
+10 -2 -5 +1
```
4를 출력

## 파트 2

주어진 입력의 숫자를 더할 때마다 나오는 숫자 중, 처음으로 두 번 나오는 숫자를 리턴하시오.

예)
```
+3, +3, +4, -2, -4
```
위 예에서는 10이 처음으로 두 번 나오는 숫자임.
0 -> 3 (+3) -> 6 (+3) -> 10(+4) -> 8(-2) -> 4(-4) -> 7(+3) -> 10(+3) -> ...
